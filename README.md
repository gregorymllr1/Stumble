# Stumble 🎲

Welcome to **Stumble**, a web project dedicated to bringing back the magic of random internet discovery. Inspired by the golden era of StumbleUpon, this site features a single button that teleports you to an interesting, unexpected corner of the web. 

## 🌟 The Vision

The internet used to be about serendipitous discovery—clicking a button and ending up on a fascinating blog, a weird web experiment, or an incredibly niche resource you didn't know you needed. Stumble aims to recreate that feeling. No endless feeds, no algorithmic echo chambers, just pure exploration.

## ✨ Features

* **The Stumble Button:** One click, infinite possibilities.
* **Curated Randomness:** A collection of fascinating websites, oddities, and hidden gems.
* **Frictionless Experience:** Clean, simple, and lightweight. You're here to explore, not navigate menus.
